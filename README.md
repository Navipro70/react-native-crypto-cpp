# react-native-cryptopp-cpp

Cryptopp implementation in react native as C++ module

## Installation

```sh
npm install react-native-cryptopp-cpp
```

## Usage


```js
import { multiply } from 'react-native-cryptopp-cpp';

// ...

const result = multiply(3, 7);
```
